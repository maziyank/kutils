# What is Kutils?

K - Collection of small but productive utilities in one place. This app hosted in https://the-kutils.web.app/

# Main Library
- svelte
- svelte-routing
- spectre.css
- lodash-es

## Get started

Clone this Repo and Install the dependencies...

```bash
git clone https://github.com/maziyank/kutils
npm install
npm run dev
```
